# gloo-project

## Introduction

This project was realized in pair for an assignement in Object-Oriented Programming. It aims at creating a playable _Sliding Block_ game in Java with the help of UML models. You have a both a version with a GUI and one which is terminal based.

## Usage

You can play the game following these steps : 

* Go inside SlidingBlock/src
* Run GraphicalInterface.java or TextInterface.java
* You are all set you can play the game for hours
